Final Project:

Shown below are a few pictures of my final project! I discussed how the base of the table was created in the entry for Week 12. I created the table tops by ironing wood veneer onto two boards of equal size. Next, I used a laser cutter programmed to use a design I had simplified in adobe acrobat. The table tops attatch to the base with velcro, so they can be interchanged with each other. Overall, I'm really happy with how this project turned out! I've learned a lot in this class, and I enjoyed getting to apply some of what I've learned to this final project.
